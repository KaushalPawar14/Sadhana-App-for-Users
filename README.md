## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/folk_app.git
```

### 2. Navigate to the project folder

```bash
cd folk_app
```

### 3. Install dependencies

```bash
flutter pub get
```

### 4. Run the app

```bash
flutter run
```
